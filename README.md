## ğŸ’¿ Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/Mohamed2007Sarhan/dr-hoda.git
    ```

2.  **Open the Project**
    simply open `index.html` in any modern web browser. No local server is required, though using VS Code's "Live Server" is recommended for the best development experience.

3.  **Customization**
    -   To change colors, modify the `:root` variables in `css/main.css`.
    -   To update content, edit the HTML files directly.
